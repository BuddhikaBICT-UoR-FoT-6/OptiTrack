# OptiTrack — AI-Powered Enterprise Resource & IoT Fleet Manager

## Overview
OptiTrack is an enterprise-grade fleet management platform that combines real-time IoT telemetry simulation, WebSocket-driven dashboards, and Gemini AI-powered analytics to deliver contextual speed/traffic risk assessment and driver scorecards.

## Tech Stack
| Layer | Technology |
|---|---|
| Frontend | React, Zustand, Tailwind CSS, Axios, STOMP.js |
| Backend | Java 21, Spring Boot 3, Spring Security (JWT), Spring Data JPA, WebSocket |
| Database | PostgreSQL |
| AI Engine | Google Gemini API |
| IoT Simulation | Spring `@Scheduled` Telemetry Scheduler |
| Testing | JUnit 5, Mockito, Selenium WebDriver (POM) |

## Project Structure
```
OptiTrack/
├── optitrack-backend/    # Spring Boot 3 application
└── optitrack-frontend/   # React + Vite application
```
